# 지수다(gisuda) — 7가지 요구사항 반영 완료

## 날짜
2026-07-11

## 요청 내용
1. 슈퍼 관리자 계정 admin, 비밀번호 `vudghkakstp``
2. 슈퍼 관리자는 일반 관리자 지정 가능
3. 스크랩 데이터에 '지역', '번역', '한줄논평' 항목 추가
4. /scraps 페이지에서 제목, 지역, 신문사, 뉴스날짜, 키워드, 요약 첫 150자만 표시, 클릭 시 상세 페이지 이동
5. 스크랩 요약 렌더링 시 줄바꿈 표시
6. 달력에 스크랩 기사의 개수가 표시
7. 게시판 목록에 본문 최초 150자 표시

## 작업 내용

### 1. 슈퍼 관리자 계정
- DB 스키마: `users.role` CHECK에 `SUPER_ADMIN` 추가
- 시드 데이터: `admin@gisuda.kr` / `vudghkakstp\`` (role=SUPER_ADMIN)
- auth.ts: SUPER_ADMIN/ADMIN/USER 3단계 역할 지원

### 2. 관리자 지정 기능
- `/api/users` PATCH 엔드포인트 추가 (SUPER_ADMIN만 역할 변경 가능)
- 관리자 페이지: 사용자 목록에 역할 변경 버튼 (승격/해제)
- SUPER_ADMIN은 "👑 슈퍼관리자", ADMIN은 "🛡️ 관리자"로 표시

### 3. 스크랩 데이터 추가 필드
- DB 스키마: `region`, `translation`, `commentary` 컬럼 추가
- 스크랩 추가 폼: 🌍 지역, 🌐 번역, 💬 한줄논평 필드 추가
- 스크랩 상세 페이지: 각 필드 별도 섹션으로 표시 (번역=파란배경, 한줄논평=노란배경+인용)
- API: GET/POST 모두 새 필드 포함

### 4. /scraps 목록 요약 표시
- 제목, 지역(🌍), 신문사, 뉴스날짜, 키워드, 요약 첫 150자(truncate) 표시
- 클릭 시 `/scraps/[id]` 상세 페이지로 이동 (Link 컴포넌트)
- `truncate(text, 150)` 함수 적용

### 5. 줄바꿈 표시
- 스크랩 상세 페이지: `whitespace-pre-wrap` 클래스로 줄바꿈 보존
- 게시판 상세 페이지: `whitespace-pre-wrap` 적용 (기존 유지)

### 6. 달력 스크랩 개수 표시
- `scrap-count-badge` CSS 클래스 추가 (파란색 원형 배지)
- 달력 셀에 날짜별 스크랩 개수 표시 (파란색 숫자 배지)
- 달력 하단 스크랩 목록에도 truncate(150) 적용

### 7. 게시판 본문 150자 표시
- `/board` 목록에 `truncate(p.content, 150)` 적용
- `whitespace-pre-wrap`으로 줄바꿈 표시
